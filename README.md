# play-ioc-lite   
>简单可控的IoC容器    

## target
- 简单易用
- IoC基本功能

## note
- 支持根据.properties配置文件或注解来加载类  
- 仅支持单例对象  

## dev 
```sh
 git clone https://github.com/datalking/play-ioc.git
 cd play-ioc/
 ./start-build-dev.sh
```
